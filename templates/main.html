Usage

Not Properly Secured, this is test version

List all movies -GET
- http://127.0.0.1:8000/api/

Movie Search by ID -GET
- http://127.0.0.1:8000/api/movie/1
- {{request.META.HTTP_HOST}}/api/movie/[MOVIE-ID]


Add movie by Title -POST
- http://127.0.0.1:8000/api/movie/morbius
- {{request.META.HTTP_HOST}}/api/movie/[MOVIE-TITLE] **(POST this request with name)


List All comments -GET
- http://127.0.0.1:8000/api/comment/
- {{request.META.HTTP_HOST}}/api/comment/ **(Returns all Movies with comments)


Add Comment by ID of Movie -POST
- http://127.0.0.1:8000/api/comment/1
- {{request.META.HTTP_HOST}}/api/comment/[MOVIE-ID] **(POST to this url with {"comment":"comment data"}) to add comment


Search comment by Title -GET
- http://127.0.0.1:8000/api/comment/movie/Titan
- {{request.META.HTTP_HOST}}/api/comment/movie/[MOVIE TITLE] **(Return the Title and comment)
