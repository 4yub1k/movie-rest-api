Usage

List all movies -GET
- http://127.0.0.1:8000/movies/?list=all

Movie Search by ID -GET
- http://127.0.0.1:8000/movies/?id=1
- {{request.META.HTTP_HOST}}/movies/?id=1


Add movie by Title -POST
- http://127.0.0.1:8000/movies/?movie=Oscar
- {{request.META.HTTP_HOST}}/movies/?movie=(movie name) **(POST this request with name)


List All comments -GET
- http://127.0.0.1:8000/comment/?list=all
- {{request.META.HTTP_HOST}}/comment/?list=all **(Returns all Movies with comments)


Add Comment by ID of Movie -POST
- http://127.0.0.1:8000/comment/
- {{request.META.HTTP_HOST}}/comment/ **(POST to this url with { "movie_id":1, "comment":"movie comments" }) to add comment


Search comment by ID -GET
- http://127.0.0.1:8000/comment/?id=1
- {{request.META.HTTP_HOST}}/comment/?id=1 **(Return the id, Title and comment)


Search comment by Title -GET
- http://127.0.0.1:8000/comment/?movie=(movie name)
- {{request.META.HTTP_HOST}}/comment/?movie=(movie name) **(Return the id, Title and comment)
